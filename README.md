# Portfólio Pessoal - Emanuel Ferreira da Silva

Este é o portfólio pessoal de Emanuel Ferreira da Silva, desenvolvido como projeto prático para a disciplina de **Fundamentos da Programação Web**. O objetivo é apresentar habilidades em desenvolvimento web, experiência profissional e projetos acadêmicos.

## Tecnologias Utilizadas
- **HTML5** – Estrutura semântica da página  
- **CSS3** – Estilização, responsividade e cards interativos  
- **JavaScript (Vanilla)** – Validação do formulário de contato sem frameworks  

## Estrutura do Projeto

/portfolio
│-- index.html # Página principal
│-- style.css # Estilo e layout
│-- script.js # Validação do formulário
│-- assets/ # Pasta opcional para imagens e ícones


## Seções da Página
- **Sobre mim**: Apresentação profissional e objetivos  
- **Formação**: Graduação, cursos e habilidades em desenvolvimento web  
- **Portfólio**: Exemplos de projetos com HTML, CSS e JavaScript  
- **Contato**: Formulário funcional com validação e feedback visual  

## Funcionalidades
- Menu de navegação fixo e responsivo  
- Cards de portfólio interativos  
- Formulário de contato com validação de campos (nome, e-mail, mensagem)  
- Feedback visual de erros e mensagem de sucesso  
- Design minimalista, responsivo e acessível
